  • Local Storage stores data in browser without any expiration date. Data is not deleted when the browser is closed. Data is available for future session.local Storage stores up to 10 megabytes of data.
  • Session storage stores data for a session. Data is deleted when the browser is closed.Data is not available for future session. Session Storage stores up to 5 megabytes of data.

  • The Global Scope is the scope that contains, and is visible in, all other scopes. In client-side JavaScript, the Global Scope is generally the web page inside which all the code is being executed.This scope includes all variables, objects, and references that are not contained within a customized scope defined by a programmer.Variables declared Globally (outside any function) have Global Scope.Global variables can be accessed from anywhere in a JavaScript program.
  • A block is a chunk of code bounded by {} . A block lives in curly braces. Anything within curly braces is a block. The block scope restricts a variable's access to the block in which it is declared.The let & const keyword facilitates the variables to be block scoped.So a variable declared in a block with let or const is only available for use within that block.
